# Muhammad Amaan - Personal Portfolio

An interactive, responsive personal portfolio website showcasing data analyst skills, education details, and real projects.

## Tech Stack
- **Framework:** React 18 + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS v3 + Vanilla CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React & React Icons

## Features
- **Simulated Terminal (CLI mode):** A Linux-style shell allowing command-line navigation and data fetch.
- **Watermark Layout:** Watermark initials, pulsing availability indicator, and modular tab-switching content.
- **Projects Grid:** Direct link actions, categories, and technical badges.
- **Skills Matrix:** Dynamic SimpleIcons logos grouped by category.
